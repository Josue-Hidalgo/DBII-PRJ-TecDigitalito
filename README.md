## Instalación Dependencias

### Docker
- [Instalar Docker Engine](https://docs.docker.com/engine/install/)

### Node (Powershell Admin)
- [Download NVM](nvmnode.com/guide/download.html)
```bash
nvm version
```
- Instalar Node
```bash
nvm install 24.11.0
```
```bash
nvm use 24.11.0
```
```bash
npm ci
```
- Instalar Angular CLI

```bash
npm install -g @angular/cli
```

## Ejecución
- En carpeta /Frontend:
```bash
npm start
```

- En carpeta /Backend:
```bash
npm run dev
```
O bien:
```bash
npm run start
```

- En carperta /: 
```bash
docker compose up 
```
